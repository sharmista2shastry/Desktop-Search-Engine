README - DESKTOP SEARCH ENGINE

In this application, a user will enter a query and the program will display the most relevant and appropriate files related to the query using tree search algorithm and giving index score to each file. The files will be then displayed according to their index score. It provides an easy way to display and search files when given key words or statements. It also provides function to create text files and search them through the directories. It is compiled in PYTHON 3.7.2.
-------------------------------------------

Software Required: Python 3.7

1. Make sure there are no folders in the directory you wish to work with.

2. Create an empty file with the name 'Reg.txt' and save it in the same place where you have saved the source code.

3. Execute DSAProject.py
